Imágen Flotante

Una imagen flotante en HTML es una imagen que se desplaza hacia un lado de la página, dejando espacio para que el texto fluya por el otro lado. Para lograr este efecto, se utiliza la propiedad float de CSS, que permite que los elementos se ubiquen a la izquierda o derecha de su contenedor. Los elementos flotantes pueden ser imágenes, tablas, cuadros de texto, gráficos y formas. Se utilizan para crear diseños web con múltiples columnas, ya que el comportamiento por defecto es que las columnas se ubiquen unas debajo de las otras.

Imágen Animada


Una imagen animada en HTML es una imagen que se crea utilizando el formato GIF (Graphics Interchange Format), que es un formato gráfico digital que permite mostrar animaciones. El formato GIF es el más utilizado para crear banners y otros elementos gráficos con animaciones. Es adecuado para representar elementos y bloques de color grandes y simples, como logotipos, texto, ilustraciones, etc. Además del GIF, HTML también soporta otros formatos de imagen, como JPEG, PNG y APNG.


Imágen como background


La propiedad background-image se emplea para establecer la imagen de fondo de cada elemento. La imagen se muestra en la zona que ocupan el contenido y el relleno del elemento, justo hasta su borde.

En HTML, la imagen de fondo es una imagen que se coloca detrás de un elemento HTML o de la página completa, utilizando la propiedad CSS background-image. La propiedad background es un atajo para especificar propiedades referentes al background en una sola regla de css. "background-color" solo sirve para el color de fondo mientras que con "background" puedes especificar más cosas
